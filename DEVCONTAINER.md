# Devcontainer usage

Open this repository in VS Code and choose: **Remote-Containers: Reopen in Container**.

If you change the Dockerfile or the devcontainer.json, rebuild the container via:

```
# from within VS Code command palette: Remote-Containers: Rebuild Container
```

Common quick checks:
- Verify Python: `python3 --version`
- Verify Node: `node --version`
